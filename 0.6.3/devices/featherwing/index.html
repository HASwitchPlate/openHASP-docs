    Adafruit Featherwing~

    You can run openHASP on the Adafruit HUZZAH32 – ESP32 Feather Board with a TFT Featherwing display. We provide pre-build firmware for this hardware combo. You can pick the 2.4" or 3.5" touchscreen.

    The TFT Featherwing comes in 2 sizes:

    • TFT FeatherWing - 2.4" 320x240 Touchscreen: ILI9341 with STMPE610 resistive touch
    • TFT FeatherWing - 3.5" 480x320 Touchscreen: HX8357D with STMPE610 resistive touch

    Development Boards~

    HUZZAH32~

    The HUZZAH32 – ESP32 Feather Boards come in 3 versions:

    • No headers soldered
    • Headers pre-soldered
    • Stacking headers pre-soldered

      Note

    Don't use the ESP8266 based Feather HUZZAH because it lacks both in memory and MCU power.

    The HUZZAH32 – ESP32 and TFT Featherwings are plug-and-play. You only need to upload the firmware via the USB port. If you get a version with pre-soldered headers then no soldering is required to get started!

    Check out the Adafruit TFT Featherwing tutorial on learn.adafruit.com. It contains more detailed information, tutorials and wiring diagrams.

    The Huzzah32 boards only have 4MB flash. If you want more storage, consider getting the SparkFun Thing Plus ESP32-WROOM instead.
